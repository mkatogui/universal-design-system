Token-Driven Styling

Every color, spacing value, radius, and shadow in this page comes from CSS custom properties. Switch palettes above to see the entire page transform.

BEM Component Naming

Components follow BEM conventions: .uds-btn, .uds-card__header, .uds-btn--primary. This keeps specificity flat and components composable.

Palette Architecture

Foundation tokens (typography, spacing, motion) are locked. Palette tokens (color, shadow, radius, display font) change per theme. One palette per surface, no mixing.
