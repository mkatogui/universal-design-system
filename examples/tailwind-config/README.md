# Tailwind CSS Config Example

A working Tailwind CSS v3 configuration generated by the Universal Design System
using the **minimal-saas** palette for a SaaS Landing Page.

## How it was generated

```bash
python src/scripts/design_system.py "saas landing page" --format tailwind
```

## Usage

1. Copy `tailwind.config.js` into your project root.
2. Install Tailwind CSS if you haven't already:
   ```bash
   npm install -D tailwindcss postcss autoprefixer
   ```
3. Use UDS tokens as Tailwind utility classes:

```html
<button class="bg-brand-primary text-white rounded-md px-4 py-2">
  Get Started
</button>

<div class="bg-bg-secondary border border-border rounded-lg shadow-md p-6">
  <h2 class="font-display text-text-primary text-xl">Feature Card</h2>
  <p class="text-text-secondary mt-2">Description text here.</p>
</div>
```
